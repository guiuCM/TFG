# TFG
This Bachelor’s thesis, developed within the framework of the Computer Engineering
degree at the Barcelona School of Informatics (FIB), develops an efficient algorithm for
generating navigation grids in complex outdoor environments.
Using an incremental GRID to TIN strategy, the system transforms height maps into
optimized triangle networks, offering a scalable solution for the navigation of intelligent
agents in video games, building upon the research work of one of the project supervisors.
